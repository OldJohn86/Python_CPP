How to program Serial-NAND flash on Saturn Eng. board(Empty flash)

Blue font means this command is on RLX console.
Red font means this command is on Saturn console.

  1. Connect JTAG to Saturn Eng. board and power on
  2. Put rlx_probe0.cfg to RLX Prober folder and update content to load BL2 image.
  3. Execute RLX Probe
  4. BL2 will perform DDR training by Type "c" on saturn console. Suppose you see "DDR Init done..." message on Saturn console
  5. Type "stop" on RLX console
  6. Type "load C:\saturn-u-boot.bin 0x84000000" on RLX console to load U-Boot
  7. Type "g==0x84000000" on RLX console
  8. Press any key on Saturn console to stop Saturn
  9. Type "stop" on RLX console
  10. Type "load P:\Work\Saturn\SSB\ca82xx-devel\gpt-tools\saturn-gpt-snand 0x85000000" on RLX console to load GPT image
  11. Type "c" on RLX console to continue
  12. Type "spi_nand erase 0 0x300000;spi_nand write 0x85000000 0 0x300000" on Saturn console
  13. Wait flash programming finish(no more than 5 seconds)
  14. Push reset button to reset Saturn board. It will stop in U-Boot CLI. Type 'c' to continue boot.

Upgrade flash image on Saturn SFU/SFP board(bootable board)

SFU: You can skip the last step(user partition) if you want to keep your local change(e.g: scfg) 

SATURN# tftpboot 0x85000000  major-image-saturn-sfu-eng-gpt.img
SATURN# spi_nand erase 0x0000000 0x00400000; spi_nand write 0x85000000  0x0000000 0x300000
SATURN# tftpboot 0x85000000  uboot-env.bin
SATURN# spi_nand erase 0x0400000 0x00100000; spi_nand write 0x85000000  0x0400000 0x20000
SATURN# tftpboot 0x85000000  uImage-saturn_sfu.dtb
SATURN# spi_nand erase 0x0500000 0x00100000; spi_nand write 0x85000000  0x0500000 0x20000
SATURN# tftpboot 0x85000000  uImage
SATURN# spi_nand erase 0x0600000 0x00600000; spi_nand write 0x85000000  0x0600000 0x500000
SATURN# tftpboot 0x85000000  major-image-saturn-sfu-eng.rootfs.squashfs.ubi
SATURN# spi_nand erase 0x0c00000 0x02800000; spi_nand write 0x85000000  0x0c00000 ${filesize}
SATURN# tftpboot 0x85000000  major-image-saturn-sfu-eng_user.ubi
SATURN# spi_nand erase 0x6300000 0x01400000; spi_nand write 0x85000000  0x6300000 ${filesize}
