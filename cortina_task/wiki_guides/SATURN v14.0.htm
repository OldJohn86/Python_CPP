AXIPREVENT    
  
When set to '1' the OTP arbiter will prevent AXI data read or write
transactions to the OTP macro.	An AXI read to the data array will
return all zeros. AXI read/writes to these registers will still
occur correctly.
 
PWR    
  
Setting PWR asserts the OTP array WAKEUP pin for at least 10us and then
asserts a RESETB pulse and finally sets CONNECT to 2'b11.
 
RESETB    
  
Soft reset signal for kilopass controller, ANDed with the main block reset_n
signal.  Note: If used, resetb should be set to '0' for at least32 LSAXI clock
cycles
